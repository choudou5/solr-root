产品功能

易接入

不改动solr核心代码，只需将Solr管理系统 集成到现有solr服务，即可享有一套丰富的界面化管理功能。

安全

系统采用 高性能的XXTea加密算法(QQ协议就是采用此算法)+填充算法 进行数据授权访问，可为每个子应用分配授权令牌。

多数据源

支持 Mysql, SqlServer, Excel数据导入，同时支持 定时增量导入。



精准分词

系统提供多种分词器：正向分词（IkAnalyzer, mmseg4j）, 正向连续分词，拼音分词，树结构分词，地区分词，自定义符号切割分词等等。

云词库

支持 在线添加模块词库，您只需要配置SQL语句即可 热生效索引数据。

故障恢复

不小心删掉了 zk或本地的 solr配置文件？ 一键恢复。
